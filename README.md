# justaprint.nvim
A simple plugin just for the sake of making one

## Features
Contains one function which prints "hello world"

## Installation
Using [lazy.nvim](https://github.com/folke/lazy.nvim)

```lua
{
    'meni/justaprint.nvim',
    config = function()
        require('justaprint.nvim')
    end
}
```
